# go-space

A structured Go learning environment — built project by project, concept by concept.

> Goal: Go from zero to building a TUI framework from scratch.

---

## Curriculum

### Beginner
| # | Project | Concepts |
|---|---------|----------|
| 1 | [calculator](./calculator) | types, functions, switch, os.Args, strconv |
| 2 | [todo](./todo) | structs, slices, methods, multiple files |
| 3 | [wordcounter](./wordcounter) | file I/O, bufio, maps, error handling |
| 4 | HTTP server *(next)* | net/http, handlers, routing |

### Intermediate
| # | Project | Concepts |
|---|---------|----------|
| 1 | REST API | interfaces, JSON, middleware |
| 2 | web scraper | goroutines, WaitGroups, channels |
| 3 | task manager | file persistence, flags package |
| 4 | rate limiter | channels, tickers, goroutine patterns |

### Advanced
| # | Project | Concepts |
|---|---------|----------|
| 1 | key-value store | maps, mutexes, context |
| 2 | worker pool | generics, channels, error aggregation |
| 3 | mini CLI framework | interfaces, reflection |
| 4 | TUI app | Elm architecture, event loops, rendering |
